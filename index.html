My Projects

XOXO Game

Purpose: A classic and engaging game of Tic Tac Toe designed to showcase my understanding of JavaScript and game logic.
Technologies Used: HTML, CSS, JavaScript.
Key Features: Two-player mode, responsive design that adapts to different screen sizes, and real-time updates of the game state.
Challenges and Solutions: Ensuring accurate game logic and handling edge cases such as draw conditions. Utilized JavaScript functions to efficiently check win and draw conditions, ensuring smooth gameplay.
Outcome: Successfully created an interactive and visually appealing Tic Tac Toe game that provides an enjoyable experience for players. The project demonstrates my ability to build functional web applications with a clean and responsive interface.
